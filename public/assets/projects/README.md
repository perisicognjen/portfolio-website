# Project images

Add screenshots here and reference them in `src/data/portfolio.json`.

| Folder     | Status   | Example filename              |
|------------|----------|-------------------------------|
| `completed/` | Completed | `ecommerce-platform.jpg`    |
| `ongoing/`   | Ongoing   | `portfolio-website.jpg`     |
| `planned/`   | Planned   | `mobile-fitness.jpg`          |

In `portfolio.json`, set `image` to e.g. `assets/projects/ongoing/portfolio-website.jpg`.

Recommended size: **1200×675** (16:9), JPG or WebP.
